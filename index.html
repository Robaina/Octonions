The algebra of octonions

I'm sure you have heard before of the complex numbers. $$p = a_0 + a_1i$$ in which $i^2 = -1$ and where the coefficients $a_0, a_1$ are real numbers. Perhaps you also have heard of the quaternions: $$p = a_0 + a_1i + a_2j + a_3k$$ with $ijk = -1$ and $i^2 = j^2 = k^2 = -1$. Now, look at this beauty: the algebra of octonions! The last normed division algebra of the Cayley - Dickson construction process. The one that it not ordered or commutative, not even associative! $$p = a_0 + a_1i + a_2j + a_3k + a_4I + a_5J + a_6K + a_7E$$ The multiplication rule here is a little bit more complicated... But it can be visualized in a beautiful way with the help of the Fano plane shown below.
The Fano plane
In the Fano plane, each circle represents one of the imaginary units in an octonion. And the lines represent the direction of multiplication. For instance, following the arrows, we have that $Ik = J$. Note that with the Fano plane — and a clever naming of the imaginary units — it is easy to see the structure of an octonion, composed of two, nested quaternions. Below is the multiplication table of the octonions which contains all the multiplication rules for the imaginary units depicted in the Fano place. With it, you should be able to have fun doing some octonion arithmetic — remember that division is equivalent to multiplying by the multiplicative inverse of a number. However, for those of you that are like me and despise doing calculations at hand, I have written a JavaScript function to do arithmetic with octonions. All you have to do is to type your octonions in the input boxes and choose which operation you would like do. Have fun!
